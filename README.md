# Chord Finder

## Description

A tool that helps musicians identify and visualize chord compositions.
Fully responsive on desktop and mobile. Feel free to try it via the Demo link.

## Demo

[Demo](https://tesheng-chordfinder.netlify.app/)

## Tech Stack

![Vue.js](https://img.shields.io/badge/vuejs-%2335495e.svg?style=for-the-badge&logo=vuedotjs&logoColor=%234FC08D) ![Bootstrap](https://img.shields.io/badge/bootstrap-%238511FA.svg?style=for-the-badge&logo=bootstrap&logoColor=white)

- Vue.js for building a reactive web application
- Bootstrap for overall layout and responsive design
- HTML, CSS, Sass, JavaScript
- Vite as the build tool

## Background

When I worked as a piano mentor, it was always difficult to explain chord
compositions during class, especially when it came up spontaneously during
lessons. So I developed this tool to help me visualize chord compositions
immediately by selecting the root note and chord type.

## Features

- Select root note and chord type
- Display notes on screen with piano keyboard UI

## Running on Local Machine

After **git clone** repo links, in the terminal

```
cd chord-finder
```

> Note: the working directory should in /chord-finder not the CHORD-FINDER

download the dependencies

```
npm install
```

running on localhost port

```
npm run dev
```
